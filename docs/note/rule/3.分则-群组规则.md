## 3.1 定义

 - **群组**：由我们创建的QQ群/Kook/TeamSpeak语音频道
 - **刷屏**：玩家在聊天公屏内发送连续多条消息

## 3.2 群组内行为

3.2.1 禁止在群内发送涉黄/涉政/涉毒/涉恐等违法不良信息

3.2.2 禁止在群内发送谣言

3.2.3 禁止在群内刷屏/对其他玩家进行攻击/谩骂/造谣

3.2.4 禁止发布他人的隐私信息

3.2.5 禁止进行与服务器玩法无关的广告宣传

3.2.6 禁止在群组内对其他玩家进行骚扰

## 3.3 处罚

我们将视情节严重程度，对违反规则的玩家除以群内禁言/踢出群聊的惩罚
